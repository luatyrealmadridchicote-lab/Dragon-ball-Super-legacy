# Dragon Ball Super Legacy - Minecraft Education Edition Addon

## 🐉 Overview

## 📦 Installation

1. **Download the addon files**
2. **Extract to Minecraft Education folder:**
   - Windows: `%appdata%\.minecraft\behavior_packs` and `resource_packs`
   - Mac: `~/Library/Application Support/minecraft/behavior_packs` and `resource_packs`
3. **Open Minecraft Education Edition**
4. **Create new world**
5. **Enable both behavior pack and resource pack in world settings**
6. **Load world and type:** `/go dragon ball`

## 🎮 How to Play

### Starting the Adventure
```
/go dragon ball
```

### Character Selection
- Choose your character during world initialization
- Each has unique abilities and story progression

### Story Progression
1. Complete tutorial with Master Roshi
2. Train to increase power level
3. Face progressively stronger opponents
4. Unlock transformations through battles
5. Progress through all Dragon Ball sagas
6. Reach the Tournament of Power finale

### Combat
- **LMB/Sneak+Jump** - Ki Blast
- **Sneak+Attack** - Special Ability
- **Crouch+Sneak** - Transformation
- **Interact with NPC** - Dialogue/Training

### Power-Up System
- Defeat enemies to gain experience
- Level up to increase base stats
- Unlock transformations at power thresholds
- Equip better gear for bonuses

## ⌨️ Commands

```
/go dragon ball              - Start adventure
/transform ssj              - Super Saiyan
/transform ssj2             - Super Saiyan 2
/transform ssj3             - Super Saiyan 3
/transform ssjblue          - Super Saiyan Blue
/transform ui               - Ultra Instinct
/ability kamehameha         - Launch Kamehameha
/ability spiritbomb         - Spirit Bomb
/ability destructodisc      - Destructo Disc
/fusion <player>            - Fusion Dance
/battle <boss_name>         - Start boss battle
/stats                      - View your stats
/powerlevel                 - Check power level
/tournament                 - Enter Tournament Mode
/questlog                   - View story missions
```

## 🐛 Troubleshooting

### Addon not showing?
- Verify both behavior pack and resource pack are enabled
- Restart Minecraft Education
- Check file extraction paths

### Commands not working?
- Ensure cheats are enabled
- Type `/reload` to refresh
- Check syntax with `/help`

### Performance issues?
- Reduce render distance
- Disable resource pack temporarily
- Lower particle effects settings

### Characters not appearing?
- Summon with `/summon goku`
- Check entity behavior pack enabled
- Verify texture pack loaded
