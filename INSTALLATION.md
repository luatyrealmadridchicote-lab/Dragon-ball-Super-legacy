# Installation Guide
# Dragon Ball Super Legacy for Minecraft Education Edition

## Installation Steps

### Windows
1. Download the addon files
2. Extract the ZIP file
3. Copy `behavior_pack` folder to:
   ```
   %appdata%\.minecraft\behavior_packs
   ```
4. Copy `resource_pack` folder to:
   ```
   %appdata%\.minecraft\resource_packs
   ```
5. Open Minecraft Education Edition
6. Create a new world
7. Go to world settings → Resource Packs → Activate Dragon Ball Resource Pack
8. Go to world settings → Behavior Packs → Activate Dragon Ball Behavior Pack
9. Create world and play!

### Mac
1. Download the addon files
2. Extract the ZIP file
3. Copy `behavior_pack` folder to:
   ```
   ~/Library/Application Support/minecraft/behavior_packs
   ```
4. Copy `resource_pack` folder to:
   ```
   ~/Library/Application Support/minecraft/resource_packs
   ```
5. Open Minecraft Education Edition
6. Create a new world
7. Enable both packs in world settings
8. Create world and play!

## Troubleshooting

### Addon not appearing
- Verify both folders are in correct locations
- Restart Minecraft Education
- Check that pack names match in world settings

### Commands not working
- Make sure cheats are enabled in world settings
- Type `/reload` to refresh the world
- Check spelling of commands

### Low FPS/Performance Issues
- Reduce render distance
- Disable unnecessary resource packs
- Lower particle effect settings

### Characters not spawning
- Use `/summon` command with entity ID
- Verify behavior pack is enabled
- Check that entity definitions are loaded

## Getting Started

Type in chat: `/go dragon ball`

This command will:
- Initialize your character
- Start the story mode
- Give you starting equipment
- Introduce you to the first NPCs
